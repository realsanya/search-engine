<section aria-label="The Meta Description in short" class="highlighted-box highlighted-box--with-background typo-block"><div aria-hidden="true" class="highlighted-box__title">
<span class="highlighted-box__title-icon"><i class="post-type-icon post-type-icon--reversed-style" style="background-image: url(/assets/images/icons/post-types/meta-description--reversed.svg)"></i></span>The Meta Description in short
	</div>
<div class="highlighted-box__content">
<p>The meta description defines the description of a page in just one or two sentences and is shown in search engine result pages. Due to this it's a major driver behind your website's performance in search engine result pages.</p>
<p>Keep the following things in mind when writing meta descriptions:</p>
<ul>
<li>Keep its length between 430 pixels and 920 pixels (± 70 and 155 characters respectively).</li>
<li>Make sure the meta description is easy to read and contains relevant keywords.</li>
<li>Prevent duplicate content issues by making the meta description unique for every indexable page.</li>
</ul>
</div>

<h2 id="what-is-a-meta-description">
<a aria-hidden="true" class="anchor-link" href="#what-is-a-meta-description" tabindex="-1"></a>What is a Meta Description?</h2>
<p>The meta description defines the description of a page in just one or two sentences, and is shown in search engine result pages. The meta description plays a big role within search engine marketing and deserves your attention.</p>
<p>The meta description is part of the <a href="/academy/meta-tags/">meta tags</a> family, HTML elements that are defined in a page's <code class="language-plaintext highlighter-rouge">&lt;head&gt;</code> section to help instruct browsers and search engines on how to behave on your page as well as how to display your content in the search engine results page.</p>
<h3 id="meta-description-poperties">
<a aria-hidden="true" class="anchor-link" href="#meta-description-poperties" tabindex="-1"></a>What properties does a Meta Description have?</h3>
<h4 id="length">Length</h4>
<p>Use these rules of thumb when it comes to the meta description length:</p>
<ul>
<li>
<strong>Maximum length:</strong> 155 characters or 920 pixels</li>
<li>
<strong>Minimum length:</strong> 70 characters or 430 pixels</li>
</ul>
<p>In most cases the lengths above work well for most devices for Google, Bing, Yahoo and DuckDuckGo. These lengths are however no guarantee that search engines display your meta description. In the end, that's up to the search engines.</p>
<h4 id="other-properties-that-make-a-meta-description-great">Other properties that make a Meta Description great</h4>
<p>Besides the right length, a great meta description has:</p>
<ul>
<li>Great readability</li>
<li>Relevant keywords</li>
<li>Delivers the message</li>
<li>Uniqueness</li>
</ul>
<h3 id="synonyms-for-meta-description">
<a aria-hidden="true" class="anchor-link" href="#synonyms-for-meta-description" tabindex="-1"></a>Synonyms for Meta Description</h3>
<p>Because of its relevance to SEO, the meta description is sometimes also called "seo description". For the sake of ease when we refer to the meta description element we'll call it meta description.</p>

<h2 id="how-do-you-write-optimal-meta-descriptions">
<a aria-hidden="true" class="anchor-link" href="#how-do-you-write-optimal-meta-descriptions" tabindex="-1"></a>How do you write optimal Meta Descriptions?</h2>
<p>Writing great meta descriptions isn't easy, that's why below we describe what makes a meta description great:</p>
<ul>
<li>Great readability - getting the point across even though there is so little space.</li>
<li>Relevant keywords - use the most relevant keywords for each page.</li>
<li>Delivering the message - a meta description should convey benefits or USPs.</li>
<li>Good length - a meta description should be between 70 - 155 characters (430 -920 pixels).</li>
<li>Uniqueness - every meta description should be unique.</li>
</ul>
<p>Before we get into detail, let's look at some examples first.</p>
<h3 id="example-ipad-4-mini">
<a aria-hidden="true" class="anchor-link" href="#example-ipad-4-mini" tabindex="-1"></a>Example: iPad 4 mini</h3>
<p>Apple's meta description for the query <code class="language-plaintext highlighter-rouge">apple ipad 4</code>:</p>
<div class="media-container" style="max-width: 548px;"><img alt="Apple's meta description for the query: apple ipad 4" class="image" height="99" loading="lazy" src="/wp-content/uploads/2016/05/image01-1.png" title="" width="548"/></div>
<ul>
<li>Readability: Apple has thought about the most important keywords for this page. And they've been used in the meta description.</li>
<li>Relevant keywords: the meta description is relevant for the search query.</li>
<li>Delivering the message: the meta description contains a Call-To-Action (CTA), namely: "Buy online, or visit an Apple store today". This is done well.</li>
<li>Length: the length is good. It's not too long and not too short.</li>
<li>Uniqueness: good. This is the only page that contains this meta description.</li>
</ul>
<h3 id="example-buy-nike-air-max">
<a aria-hidden="true" class="anchor-link" href="#example-buy-nike-air-max" tabindex="-1"></a>Example: Buy Nike Air Max</h3>
<p>Two listings for the query 'buy nike air max':</p>
<h4 id="the-official-nikecom-website">The official Nike.com website</h4>
<div class="media-container" style="max-width: 529px;"><img alt="Meta description of the official Nike.com website" class="image" height="77" loading="lazy" src="/wp-content/uploads/2016/05/image05-1.png" title="" width="529"/></div>
<p>Our assessment of this meta description:</p>
<ul>
<li>Readability: good.</li>
<li>Relevant keywords: reasonable, although they could have included the word 'buy'</li>
<li>Delivering the message: not good. Their meta description is dull, and does not speak to the imagination of the users.</li>
<li>Length: too short. The entire extra line is unused.</li>
<li>Uniqueness: good.</li>
</ul>
<h4 id="e-commerce-website-footasylumcom">E-commerce website Footasylum.com</h4>
<div class="media-container" style="max-width: 561px;"><img alt="Meta description of e-commerce website Footasylum.com" class="image" height="98" loading="lazy" src="/wp-content/uploads/2016/05/image02-1.png" title="" width="561"/></div>
<p>Our assessment of this meta description:</p>
<ul>
<li>Readability: good.</li>
<li>Relevant keywords: good</li>
<li>Delivering the message: very good. They speak to the imagination of the users with "Nike Air Max trainers are one of the most iconic Nike trainers in Nike's catalogue"</li>
<li>Length: very good.</li>
<li>Uniqueness: good.</li>
</ul>
<p>The title for this page is not well written, but Footasylum's meta description sure beats Nike.com's.</p>
<h3 id="great-readability">
<a aria-hidden="true" class="anchor-link" href="#great-readability" tabindex="-1"></a>Great readability</h3>
<p>As with the <a href="/academy/title-tag/">title tag</a>, the meta description should be written with the user in mind. For a user the meta description is a deciding factor in whether or not he clicks on your listing.</p>
<p>Avoid:</p>
<ul>
<li>Meta descriptions tags written in CAPITALS or lowercase only. Mix it and use uppercase letters to emphasize important words within the meta description. But Don't Start Every Word With a Capital;</li>
<li>Using special characters such as stars (<code class="language-plaintext highlighter-rouge">☆</code>), copyright symbol (<code class="language-plaintext highlighter-rouge">©</code>), registered trademark icon (<code class="language-plaintext highlighter-rouge">®</code>) or trademark (<code class="language-plaintext highlighter-rouge">™</code>) and such. Search Engine don't like them, and they may disregard your meta description altogether;</li>
<li>Excessive use of punctuation characters such as dots (<code class="language-plaintext highlighter-rouge">.</code>), commas (<code class="language-plaintext highlighter-rouge">,</code>), dashes (<code class="language-plaintext highlighter-rouge">-</code>) and  pipes (<code class="language-plaintext highlighter-rouge">|</code>).</li>
</ul>

<h3 id="relevant-keywords">
<a aria-hidden="true" class="anchor-link" href="#relevant-keywords" tabindex="-1"></a>Relevant keywords</h3>
<p>Each page should have a relevant meta description which describes what people can expect on the page. Search engines often ignore irrelevant meta description and come up with a description of their own. You can however do your best to prevent this by defining relevant meta description. In order to this this, it's important for you to know for each page what the most important keywords are. You can then query those keywords and see if your manually crafted meta description is used or not. If it's not used, you need to re-work your meta description.</p>
<p>Search engines display the visitor's query in bold within the URL, title and the description of the listing. This differs per search engine, Google only bolds keywords in the URL and description whereas Bing bolds keywords in the title as well. Bolding keywords is a way for Google and Bing to indicate the relevance between the search keywords and the results, therefore it's important to use the most important keywords in the description.</p>
<p>Using the most important keywords in the meta description not directly influences your rankings, although it has an interesting indirect impact: when visitors see bolded keywords in results they tend to click on those results more. This increases your click through rate (CTR), which in turns plays a role in determining the rank for your page. A high CTR is seen by search engines as a signal that a page is very relevant for a query.</p>
<p>Please note that search engines can be stubborn. Regardless of the changes you make sometimes they'll ignore your meta description. If that happens, just change the meta description from time to time and see if it gets picked up by search engines.</p>
<h3 id="delivering-the-message">
<a aria-hidden="true" class="anchor-link" href="#delivering-the-message" tabindex="-1"></a>Delivering the message</h3>
<p>Visitors need encouragement for them to convert. Often it's as simple as asking them to take action. On websites this is usually done with micro-copy and buttons that stand out and contain a message that influence a visitor to buy a product, send an e-mail or leave their phone number.</p>
<p>When it comes to meta descriptions it's important to ask a visitor to take action. Take another look at the Apple iPad 4 mini example: "Buy online, or visit an Apple store today". People are told to buy the iPad 4 online, or either go to a physical store to make the purchase. Today. That's a pretty good CTA right there.</p>
<p>Incorporating a CTA is one, but it's just as important to communicate why a user should a visitor should take action. In Apple's example this is supported by stating that the iPad mini 4 is available in Silver, Gold or Space Grey, and that you can also choose for different storage sizes. Apple communicates the benefit of choice here.</p>
<h3 id="good-length">
<a aria-hidden="true" class="anchor-link" href="#good-length" tabindex="-1"></a>Good length</h3>
<p>As with the title tag, there's no perfect length for the meta description either. The same parameters play a role: there are different search engines, different operating systems and different devices. And if your meta description is not too long or too short, there's still no guarantee the search engines displays it.</p>
<p>Don't panic though, below we explain how you improve your chances of search engines using your meta description in the SERPs.</p>
<h4 id="characters-vs-pixels">Characters vs. Pixels</h4>
<p>Regardless of the search engine, in each SERP there's limited amount of space for the meta description. Search engines Bing, Yahoo and DuckDuckGo advise to use a maximum amount of characters. Google however advises to use a maximum amount of pixels. This makes sense considering this is more accurate than characters. For instance, it takes into consideration that a character such as "P" takes up much more space than "l".</p>
<h4 id="meta-description-length-in-google">Meta Description length in Google</h4>
<p>A listing in Google's SERPs has a maximum width of 600 pixels. The meta description takes up four lines in the search engine listing, so in theory you have 2,400 pixels of space. Taking into account bolding of keywords, the fact that Google rarely (if at all) let us use these 2.400 pixels and some margin of error, we advise you to stick to a maximum length of 920 pixels. This generally translates to 155 characters.</p>
<p>Make sure your meta descriptions aren't too short either! In this case, Google may produce a description of their own. In order to prevent this as much as possible, make sure your meta description tag is at least 70 characters, or 430 pixels long.</p>
<h5 id="in-summary-meta-description-length-for-google">In summary: Meta Description length for Google</h5>
<ul>
<li>Maximum amount: 155 characters or 920 pixels</li>
<li>Minimum amount: 70 characters or 430 pixels</li>
</ul>
<p>Please note that Google's tablet and mobile versions have a greater maximum width for the meta description, however it's not possible to set specific meta descriptions for different devices, so we recommend to stick to the maximum width for desktop to make sure the meta description adheres to Google's constraints as much as possible.</p>
<p>Bing,Yahoo and DuckDuckGo don't have different rules per device regarding the maximum length of the meta description.</p>
<h4 id="meta-description-length-in-bing">Meta Description length in Bing</h4>
<p><a href="https://www.bing.com/webmaster/help/webmaster-guidelines-30fba23a" rel="noopener" target="_blank"><span class="extlink">Bing</span><span class="show-for-sr"> (opens in a new tab)</span></a> advises to use a maximum meta description length of 160 characters. Our own research has shown that meta descriptions with 160 or more characters are often cut-off in Bing, so we advise to keep it to a maximum of 155 characters when you're optimizing for Bing specifically.</p>
<p>Our advice regarding the minimum amount of characters for Bing is: 70 characters.</p>
<h4 id="meta-description-length-in-yahoo">Meta Description length in Yahoo</h4>
<p>Yahoo does not give much <a href="https://help.yahoo.com/kb/web-page-title-description-incorrect-sln2195.html" rel="noopener" target="_blank"><span class="extlink">advice in their Help section</span><span class="show-for-sr"> (opens in a new tab)</span></a> about the minimum and maximum length of the meta description. Our own research has shown that the Yahoo SERP allows for a maximum width of 553 pixels. Since the meta description consists of two lines, this roughly translates to 160 characters. However, we haven't seen this full width being used, nor have we seen meta descriptions with this many characters being displayed. In practice a significant amount of meta descriptions tags above 150 characters are truncated so we advise you to stick to a maximum amount of 150 characters.</p>
<p>Regarding the minimum amount of characters, our advice for Yahoo is: 70 characters.</p>
<h4 id="meta-description-length-in-duckduckgo">Meta Description length in DuckDuckGo</h4>
<p>Although DuckDuckGo is not very informative about their advice on how to optimize pages for their search engine, our research has shown that when you stick to a maximum of 155 characters you should be safe. Our advice regarding the minimum amount of characters for DuckDuckGo is: 70 characters.</p>
<h3 id="unique-meta-description-throughout-website">
<a aria-hidden="true" class="anchor-link" href="#unique-meta-description-throughout-website" tabindex="-1"></a>Unique Meta Descriptions throughout a website</h3>
<p>In order for search engines and users to be able to distinguish between pages, it's vital that each indexable page has a unique meta description. Both search engines and users may think that pages are duplicates of one another if they contain the same meta description. For search engines, having duplicate meta descriptions is a signal that these pages may be duplicate pages and should therefore not be included in the search results.</p>

<h2 id="why-should-you-be-using-the-meta-description">
<a aria-hidden="true" class="anchor-link" href="#why-should-you-be-using-the-meta-description" tabindex="-1"></a>Why should you be using the Meta Description?</h2>
<p>The meta description is essentially your sales pitch. You need it to be great. The meta descriptions plays a major role in a visitor decision to choose which result to click on.</p>
<h3 id="search-engine-optimization">
<a aria-hidden="true" class="anchor-link" href="#search-engine-optimization" tabindex="-1"></a>Search Engine Optimization</h3>
<p>Even though there is no <em>direct</em> relation between great meta descriptions and higher rankings, meta descriptions play a vital part in Search Engine Optimization:</p>
<ol>
<li>Meta descriptions are prominently shown in SERPs and play a big role within Search Engine Optimization.</li>
<li>Search engines track the amount of users that click on your listing in the SERPs. When a lot of visitors click through to your website from the SERP your rankings may improve over time.</li>
</ol>
<h3 id="user-experience">
<a aria-hidden="true" class="anchor-link" href="#user-experience" tabindex="-1"></a>User Experience</h3>
<p>When it comes to meta descriptions the interests of search engines and users are very much aligned. If the written meta descriptions <a href="/academy/ux-seo/#winning-clicks">works well for users</a>, the benefits from a search engine optimization point of view follow naturally.</p>
<h4 id="meta-description-heavily-influences-ctr">Meta Description heavily influences CTR</h4>
<p>Meta descriptions heavily influence click-through-rates (CTRs - the percentage of people clicking on your listing) of a listing. The meta description is essentially your sales pitch, comparable with advertisement descriptions in Google AdWords.</p>
<p>When the meta description is missing, too long or too short, search engines may produce a description of their own. Often this is sub-optimal, as you want it to look as well as possible to entice people to visit your website.</p>
<h3 id="social-media">
<a aria-hidden="true" class="anchor-link" href="#social-media" tabindex="-1"></a>Social Media</h3>
<p>In order to drive traffic from social media it's vital to use social media markup such as for instance Facebook's Open Graph and Twitter's Twitter Cards. This markup is used to retrieve information such as the title, description and images which should be shown on social media. Carefully choosing what social media platforms show when your page is shared leads to more visitors.</p>
<p>If social media markup is missing, the meta description is often used as the fall-back. So if you're unable to use social media markup, having a great meta description helps drive traffic from social media.</p>

<h3 id="how-long-should-my-meta-description-be">
<a aria-hidden="true" class="anchor-link" href="#how-long-should-my-meta-description-be" tabindex="-1"></a>1. How long should my Meta Description be?</h3>
<p>There's no perfect meta description length, but generally these lengths work very well:</p>
<ul>
<li>Maximum length: <strong>155 characters or 920 pixels</strong>
</li>
<li>Minimum length: <strong>70 characters or 430 pixels</strong>
</li>
</ul>
<p>In most cases these lengths are valid for <em>most devices</em> for Google*.</p>
<p>*: These lengths are however no guarantee that search engines display your meta description. In the end, that's up to the search engines to decide, but you can influence this by applying <a href="/academy/meta-description/#how-do-you-write-optimal-meta-description">the best practices around writing meta descriptions</a>.</p>
<h3 id="does-having-fancy-characters-in-my-meta-description-increase-my-ctr">
<a aria-hidden="true" class="anchor-link" href="#does-having-fancy-characters-in-my-meta-description-increase-my-ctr" tabindex="-1"></a>2. Does having fancy characters in my Meta Descriptions increase my CTR?</h3>
<p>There's a reason why you see them rarely: search engines don't like them. Avoid stuffing your meta descriptions with special characters such as stars (<code class="language-plaintext highlighter-rouge">☆</code>), copyright symbol (<code class="language-plaintext highlighter-rouge">©</code>), registered trademark icon (<code class="language-plaintext highlighter-rouge">®</code>) or trademark (<code class="language-plaintext highlighter-rouge">™</code>) and such.</p>
<p>Even excessively using regular special characters such as dashes (<code class="language-plaintext highlighter-rouge">-</code>), pipes (<code class="language-plaintext highlighter-rouge">|</code>) or commas (<code class="language-plaintext highlighter-rouge">,</code>) can trigger your manually defined meta descriptions not to be shown.</p>
<h3 id="should-i-manually-define-all-of-my-meta-descriptions">
<a aria-hidden="true" class="anchor-link" href="#should-i-manually-define-all-of-my-meta-descriptions" tabindex="-1"></a>3. Should I manually define all of my Meta Descriptions?</h3>
<p>That depends. In the perfect world, each and every page within your website has a manually crafted meta description. For a website with a hundred pages this isn't an issue, but for e-commerce websites with thousands of pages this is a full-time job, and the return on this huge investment over using generated meta descriptions is very uncertain.</p>
<p>Define manual meta descriptions where you can really make them count. Determine which pages are most important to you, and define manual meta descriptions for these. Criteria which help here are: potential revenue, potential organic traffic, the degree in which these pages are vital to your website (homepage, product category page, hubpages etc.).</p>
<p>Meta descriptions are often generated for <a href="/academy/product-page-seo/">product pages</a> and blog archive pages (category, author, tags, date) and so on. Make sure to write a good template which adheres to the general guidelines.</p>
<h4 id="pro-tip">Pro tip:</h4>
<p>In most cases Pareto's principle applies: 20% of the pages make up 80% the organic traffic.</p>
<h3 id="why-are-the-search-engines-still-showing-the-old-one">
<a aria-hidden="true" class="anchor-link" href="#why-are-the-search-engines-still-showing-the-old-one" tabindex="-1"></a>4. I updated my Meta Description, why are the search engines still showing the old one?</h3>
<p>Chances are search engines haven't visited your page yet since you've made changes to your meta description. Search engines need a bit of time to re-visit your page and process all the changes.</p>
<p>You wonder how long that'll take right? This varies per page. Authoritative pages and frequently updated pages are visited more often than less authoritative pages and less frequently updated pages. Search engines may pick up on the new meta description within 2 minutes but it may also take several days or even more than a week.</p>
<h3 id="can-search-engines-choose-to-show-the-description-from-the-dmoz-or-yahoo-directory">
<a aria-hidden="true" class="anchor-link" href="#can-search-engines-choose-to-show-the-description-from-the-dmoz-or-yahoo-directory" tabindex="-1"></a>5. Can search engines choose to show the description from Yahoo! Directory?</h3>
<p>Yes, they sure can. Yahoo! Directory can be part of their fall-back plans when they deem your meta description unfit. If you don't want this, or if you are not satisfied with the description in the Yahoo Directory you can set up the tags below in your <code class="language-plaintext highlighter-rouge">&lt;HEAD&gt;</code>-section:</p>
<section aria-label="A fragment of code" class="code-box"><div class="code-box__code">
<figure class="highlight"><pre><code class="language-text" data-lang="text">&lt;meta name="robots" content="noodp" /&gt; and &lt;meta name="robots" content="noydir" /&gt;</code></pre></figure>
